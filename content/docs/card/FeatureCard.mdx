## Installation

### CLI Installation

Make sure ShadCN UI and your custom UI system are set up in your project. Then run:

```bash
npx shadcn@latest add https://ui-chaiui.vercel.app/feature-card.json
```

### üßæ 3. Paste the Component Code

Paste the following code into `FeatureCard.tsx`:

```tsx 
import React from "react";

interface FeatureCardProps {
  title?: string;
  description?: string;
  icon?: string;
  ChaiTheme?: boolean;
}

function FeatureCard({
  title = "Feature Title",
  description = "Default feature description that explains what this feature does in detail.",
  icon = "‚ú®",
  ChaiTheme = false
}: FeatureCardProps) {
  return (
    <div
      className={`${
        ChaiTheme
          ? "bg-amber-50 dark:bg-amber-900 text-amber-900 dark:text-amber-50"
          : "bg-white dark:bg-neutral-900 text-neutral-800 dark:text-neutral-200"
      } w-60 rounded-2xl shadow-md p-5 flex flex-col items-center justify-center`}
    >
      <div className="text-3xl mb-2">{icon}</div>
      <div className="text-lg font-bold mb-1">{title}</div>
      <div
        className={`text-sm mb-4 ${
          ChaiTheme
            ? "text-amber-700 dark:text-amber-300"
            : "text-gray-500 dark:text-neutral-400"
        }`}
      >
        {description}
      </div>
    </div>
  );
}

export default FeatureCard;
```

## Usage

Import and use the `FeatureCard` component like this:

```tsx

        title="Regular Feature"

## Props

| **Prop**       | **Type**    | **Description**                                      | **Default**                                                  |
|----------------|-------------|------------------------------------------------------|--------------------------------------------------------------|
| `title`        | `string`    | Title text shown at the top of the card              | `"Feature Title"`                                            |
| `description`  | `string`    | Description text shown below the title                | `"Default feature description that explains what this feature does in detail."` |
| `icon`         | `string`    | Icon to be displayed at the top of the card         | `"‚ú®"`                                                       |
| `ChaiTheme`    | `boolean`   | Whether to apply Chai-themed amber color styles      | `false`                                                      |
```
